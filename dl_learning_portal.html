What is Deep Learning?
Neurons, layers and how machines learn to see, hear and understand
The Neuron

Each artificial neuron takes weighted inputs, sums them, adds a bias, then passes the result through an activation function that decides whether to "fire". Inspired by biological brain cells.

Layers

Input layer receives raw data. Hidden layers learn increasingly abstract features. Output layer produces the final prediction. "Deep" = many hidden layers.

Backpropagation

After a prediction, the error is propagated backwards through the network. Each weight is nudged in the direction that reduces error. Repeat thousands of times → the network gets smarter.

Why "Deep" Works

Layer 1 detects edges, layer 2 shapes, layer 3 objects. Each layer builds on the last. This hierarchical feature learning is what makes deep nets dramatically outperform shallow ones on complex tasks.

ANN — Artificial Neural Network
The foundation: fully connected layers, activation functions & backpropagation

🔗 What is an ANN? Foundation

An Artificial Neural Network is a series of fully connected layers. Every neuron in one layer connects to every neuron in the next. It's the building block for all other deep learning architectures.

  • Activation functions: ReLU, Sigmoid, Tanh — add non-linearity so the network can learn complex patterns
  • Loss function: Measures how wrong the prediction is (MSE, Cross-Entropy)
  • Optimiser: Adam, SGD — adjusts weights to minimise loss

Python — Single Neuron
import numpy as np

# Inputs and weights
x = np.array([1.0, 0.5])
w = np.array([0.8, 0.6])
b = 0.1

# Weighted sum + bias
z = np.dot(w, x) + b
print(f"Weighted sum z = {z:.4f}")

# Activation functions
relu    = max(0, z)
sigmoid = 1 / (1 + np.exp(-z))
tanh    = np.tanh(z)

print(f"ReLU:    {relu:.4f}")
print(f"Sigmoid: {sigmoid:.4f}")
print(f"Tanh:    {tanh:.4f}")

Python — ANN for XOR (NumPy)
import numpy as np

# XOR dataset
X = np.array([[0,0],[0,1],[1,0],[1,1]])
y = np.array([[0],[1],[1],[0]])

# Weights
np.random.seed(42)
W1 = np.random.randn(2, 4)  # 2 inputs → 4 hidden
W2 = np.random.randn(4, 1)  # 4 hidden → 1 output
b1 = np.zeros((1, 4))
b2 = np.zeros((1, 1))

sigmoid = lambda z: 1/(1+np.exp(-z))
lr = 0.5

for epoch in range(1000):
    # Forward pass
    A1 = sigmoid(X @ W1 + b1)
    A2 = sigmoid(A1 @ W2 + b2)
    
    # Loss (MSE)
    loss = np.mean((y - A2)**2)
    
    # Backprop
    dA2 = -2*(y-A2)*A2*(1-A2)
    dW2 = A1.T @ dA2
    dA1 = (dA2 @ W2.T)*A1*(1-A1)
    dW1 = X.T @ dA1
    
    # Update
    W2 -= lr*dW2;  W1 -= lr*dW1
    b2 -= lr*dA2.mean(0)
    b1 -= lr*dA1.mean(0)

print(f"Final Loss: {loss:.5f}")
print("Predictions:", A2.round(2))

CNN — Convolutional Neural Network
How machines see: filters, feature maps, pooling and image classification

🖼️ What is a CNN? Vision

A CNN slides small filters (kernels) across an image to detect local features — edges, textures, shapes. Unlike ANN, it preserves spatial structure. Used in face recognition, medical imaging, self-driving cars.

  • Convolution: Filter slides over input, computing dot products → feature map
  • Pooling: Down-samples maps (Max Pool takes the largest value in each region)
  • Flatten + Dense: Final classification layer

Python — CNN with Keras
from tensorflow.keras import layers, models
import numpy as np

# Build CNN for 28×28 grayscale images (e.g. MNIST digits)
model = models.Sequential([
    # Conv layer 1: 32 filters, 3×3 kernel, ReLU activation
    layers.Conv2D(32, (3,3), activation='relu',
                  input_shape=(28,28,1)),
    layers.MaxPooling2D((2,2)),   # halves spatial dims

    # Conv layer 2: 64 filters
    layers.Conv2D(64, (3,3), activation='relu'),
    layers.MaxPooling2D((2,2)),

    layers.Flatten(),            # flatten to 1D
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')  # 10 digit classes
])

model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

model.summary()
# Total params: ~93,000
# Achieves ~99% accuracy on MNIST

RNN — Recurrent Neural Network
Networks with memory: sequences, time-series and text generation

🔄 What is an RNN? Sequence

Unlike ANN/CNN, RNNs have loops — they pass a hidden state from one time step to the next, giving them a form of memory. Perfect for anything that unfolds in time: text, audio, stock prices.

  • Hidden state h: Carries information from previous steps
  • Vanishing gradient: Classic RNNs forget distant context — solved by LSTM/GRU
  • Uses: Text generation, speech recognition, sentiment analysis, time-series prediction

Python — RNN for Time-Series (Keras)
from tensorflow.keras import layers, models
import numpy as np

# Create noisy sine-wave dataset
t = np.linspace(0, 50, 500)
signal = np.sin(t) + np.random.normal(0, 0.1, 500)

# Build sliding-window sequences (lookback=7)
lookback = 7
X, y = [], []
for i in range(lookback, len(signal)):
    X.append(signal[i-lookback:i])
    y.append(signal[i])
X = np.array(X)[..., np.newaxis]  # shape: (493, 7, 1)

# Build RNN model
model = models.Sequential([
    layers.SimpleRNN(32, activation='tanh',
                     input_shape=(lookback, 1)),
    layers.Dense(1)  # predict next value
])
model.compile(optimizer='adam', loss='mse')
model.fit(X, y, epochs=20, batch_size=32, verbose=0)

# Predict next step
last_seq = signal[-lookback:][np.newaxis, :, np.newaxis]
prediction = model.predict(last_seq)
print(f"Next value: {prediction[0,0]:.4f}")

LSTM — Long Short-Term Memory
Gates that selectively remember and forget — solving the vanishing gradient problem

🧬 What is LSTM? Long Memory

LSTM solves the vanilla RNN's "forgetting" problem with three gates: a forget gate (what to discard), an input gate (what new info to store), and an output gate (what to pass forward). This lets it remember context over hundreds of steps.

  • Cell state (c): Long-term memory — carried forward with minimal modification
  • Hidden state (h): Short-term output — passed to next cell and output layer
  • GRU: Simpler variant with just 2 gates, often equally effective

Python — LSTM Cell (NumPy manual implementation)
import numpy as np

sigmoid = lambda x: 1/(1+np.exp(-x))

# Previous states
h_prev = np.array([0.3])   # previous hidden state
c_prev = np.array([0.5])   # previous cell state
x_t    = np.array([0.8])   # current input

# (Simplified — weights normally learned via backprop)
# Forget gate: how much of c_prev to keep?
f = sigmoid(0.9)    # 0.71 → keep 71% of old memory

# Input gate: how much new info to add?
i = sigmoid(1.2)    # 0.77
g = np.tanh(0.6)    # candidate: 0.54

# Output gate: what to expose as hidden state?
o = sigmoid(1.5)    # 0.82

# New cell state (long-term memory)
c_new = f * c_prev + i * g
print(f"New cell state c: {c_new[0]:.4f}")

# New hidden state (output)
h_new = o * np.tanh(c_new)
print(f"New hidden state h: {h_new[0]:.4f}")

Transformer — Attention is All You Need
The architecture powering GPT, Claude, BERT — self-attention explained

⚡ What is a Transformer? State of the Art

Transformers replaced RNNs for most NLP tasks. Instead of processing tokens sequentially, they use self-attention — every token looks at every other token simultaneously, weighting how relevant each is. This enables massive parallelism and long-range dependencies.

  • Query, Key, Value (QKV): Each token creates 3 vectors. Attention = softmax(Q·Kᵀ/√d)·V
  • Multi-head attention: Run attention multiple times in parallel (different "views")
  • Positional encoding: Adds position info (since there's no recurrence)
  • GPT: Decoder-only. BERT: Encoder-only. T5: Encoder-Decoder

Python — Self-Attention (NumPy)
import numpy as np

def self_attention(X, d_k=64):
    """
    X: input embeddings shape (seq_len, d_model)
    d_k: key/query dimension
    """
    # Learned weight matrices (simplified)
    W_Q = np.random.randn(X.shape[1], d_k)
    W_K = np.random.randn(X.shape[1], d_k)
    W_V = np.random.randn(X.shape[1], d_k)

    # Compute Q, K, V projections
    Q = X @ W_Q   # Queries
    K = X @ W_K   # Keys
    V = X @ W_V   # Values

    # Scaled dot-product attention
    scores = Q @ K.T / np.sqrt(d_k)  # (seq, seq)

    # Softmax → attention weights
    e_scores = np.exp(scores - scores.max(axis=-1, keepdims=True))
    weights   = e_scores / e_scores.sum(axis=-1, keepdims=True)

    # Weighted sum of values
    output = weights @ V
    return output, weights

# Example usage
tokens = ["The", "cat", "sat", "on", "the", "mat"]
X = np.random.randn(len(tokens), 128)  # random embeddings
out, attn = self_attention(X)
print("Attention matrix shape:", attn.shape)  # (6, 6)
