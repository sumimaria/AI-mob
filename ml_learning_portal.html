What is Machine Learning?
The big picture — how machines learn from data
Traditional Programming

Rules + Data → Output

A programmer writes every rule manually. If income > 50000 AND age > 25 → approve loan. Fragile. Doesn't scale.

Machine Learning

Data + Answers → Rules

You show the machine thousands of examples. It figures out the rules itself. No manual coding of logic.

The Learning Loop

Data → Model → Prediction → Error → Adjust → Repeat. Each cycle the model gets a little better, like practising a skill.

Real-World Uses

Spam filters, face recognition, Netflix recommendations, medical diagnosis, self-driving cars, language translators.

02 🏷️
Supervised Learning
Learning from labelled examples — Regression & Classification

🏷️ What is Supervised Learning? Supervised

In supervised learning, the machine is given labelled data — every example has an input and a known correct output. It learns to map inputs → outputs, then makes predictions on new, unseen data.

  • Analogy: A student practising past exam papers with an answer key
  • Input: Features (size, colour, weight, text…)
  • Output: A label (category) or a number (prediction)

📈 Regression — Predicting a Number

Regression finds a mathematical relationship between input features and a continuous number output. "What price should this house sell for?"

  • Input: House size in sq ft, bedrooms, location score
  • Output: Predicted price in $
  • Algorithm: Linear Regression draws the best-fit line through data

Python — Linear Regression 
import numpy as np
from sklearn.linear_model import LinearRegression

# Training data (features: size, beds, location, age)
X_train = np.array([
    [800,  2, 4, 20],   # $180,000
    [1200, 3, 5, 10],   # $260,000
    [2000, 4, 7, 5],    # $420,000
    [3500, 5, 9, 2],    # $680,000
])
y_train = np.array([180000, 260000, 420000, 680000])

# Train the model
model = LinearRegression()
model.fit(X_train, y_train)

# Predict for a new house
new_house = np.array([[1500, 3, 5, 10]])
price = model.predict(new_house)
print(f"Predicted Price: ${price[0]:,.0f}")
# Output: Predicted Price: $285,000

🗂️ Classification — Predicting a Category

Classification predicts which category or class an input belongs to. Instead of a number, the output is a label like "Spam / Not Spam" or "Cat / Dog."

  • Binary: Two classes (Spam vs Ham, Fraud vs Legit)
  • Multi-class: Many classes (Dog / Cat / Bird / Fish)
  • Algorithm: Logistic Regression, SVM, Random Forest

Python — Logistic Regression Classifier 
from sklearn.linear_model import LogisticRegression
from sklearn.feature_extraction.text import TfidfVectorizer

# Sample emails and labels (0=ham, 1=spam)
emails = [
    "Meeting at 3pm today",           # 0 - ham
    "YOU WON $1,000,000 CLICK NOW",    # 1 - spam
    "Your order has shipped",           # 0 - ham
    "FREE FREE FREE limited offer!!!",  # 1 - spam
]
labels = [0, 1, 0, 1]

# Convert text to numbers (TF-IDF features)
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(emails)

# Train classifier
clf = LogisticRegression()
clf.fit(X, labels)

# Classify new email
new_email = ["Claim your FREE prize NOW!!"]
X_new = vectorizer.transform(new_email)
prediction = clf.predict(X_new)
print("SPAM" if prediction[0] == 1 else "HAM")
# Output: SPAM

🌳 Decision Trees — If/Else Learned from Data

A Decision Tree splits data using the most informative feature at each step. Each branch is a condition; each leaf is a prediction. Interpretable and powerful.

  • Root node: First, most important split
  • Branches: Conditions (age < 30, income > 50k)
  • Leaves: Final predictions (Approved / Denied)

Python — Decision Tree Classifier 
from sklearn.tree import DecisionTreeClassifier, export_text
import numpy as np

# Features: [credit_score, income_k, loan_k, employment]
# employment: 0=new, 1=self, 2=stable
X_train = np.array([
    [720, 80,  100, 2],  # approved
    [550, 35,  200, 0],  # denied
    [680, 60,  150, 2],  # approved
    [490, 25,  300, 1],  # denied
    [760, 120, 80,  2],  # approved
])
y_train = np.array([1, 0, 1, 0, 1])  # 1=approved, 0=denied

# Train tree (max_depth=3 keeps it interpretable)
dt = DecisionTreeClassifier(max_depth=3)
dt.fit(X_train, y_train)

# Print the tree rules
print(export_text(dt, feature_names=[
    'credit', 'income', 'loan', 'employment'
]))

# Predict for new applicant
applicant = [[650, 60, 150, 2]]
result = dt.predict(applicant)
print("APPROVED" if result[0] else "DENIED")

Unsupervised Learning
Finding hidden structure in unlabelled data

🔍 What is Unsupervised Learning? Unsupervised

No labels, no teacher. The algorithm explores raw data and discovers hidden patterns or groupings entirely on its own — like sorting a pile of coins by size and colour without being told the denominations.

  • Clustering: Group similar items (K-Means, DBSCAN)
  • Dimensionality Reduction: Compress data (PCA, t-SNE)
  • Anomaly Detection: Find outliers (Isolation Forest)

🫧 K-Means Clustering

K-Means assigns each data point to the nearest cluster centre (centroid), then recalculates centroids, and repeats until stable. You choose K (number of clusters).

Python — K-Means Clustering 
from sklearn.cluster import KMeans
import numpy as np
import matplotlib.pyplot as plt

# Generate customer data (annual spend vs visit frequency)
np.random.seed(42)
X = np.random.randn(100, 2) * 2  # 100 customers, 2 features

# Fit K-Means with K=3 clusters
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(X)

# Get cluster labels and centroids
labels = kmeans.labels_           # which cluster each customer belongs to
centroids = kmeans.cluster_centers_  # centre of each cluster

# Visualise
colors = ['#38bdf8', '#a78bfa', '#34d399']
for i in range(3):
    mask = labels == i
    plt.scatter(X[mask, 0], X[mask, 1],
                c=colors[i], label=f'Cluster {i}')

plt.scatter(*centroids.T, marker='X', s=200,
            c='white', zorder=5, label='Centroids')
plt.legend()
plt.title('Customer Segments (K=3)')
plt.show()

The model learns what "normal" looks like from unlabelled data. Any data point that deviates significantly is flagged as an anomaly — like a fraudulent transaction or a broken machine sensor.

Python — Isolation Forest Anomaly Detection 
from sklearn.ensemble import IsolationForest
import numpy as np

# Normal transaction features: [amount, hour, is_foreign, daily_count]
normal_transactions = np.array([
    [25,  10, 0, 3],
    [60,  14, 0, 4],
    [120, 12, 0, 2],
    [45,  16, 1, 1],  # occasional foreign OK
    # ... 1000s more normal transactions
])

# Train the anomaly detector on NORMAL data only
clf = IsolationForest(contamination=0.05, random_state=42)
clf.fit(normal_transactions)

# Check a suspicious transaction
suspicious = np.array([[8500, 3, 1, 15]])
# $8500, 3am, foreign, 15 transactions today

result = clf.predict(suspicious)
score  = clf.score_samples(suspicious)

print("🚨 ANOMALY" if result[0] == -1 else "✅ Normal")
print(f"Anomaly score: {score[0]:.3f}")
# Output: 🚨 ANOMALY  |  Anomaly score: -0.842

📉 Principal Component Analysis (PCA)

PCA compresses high-dimensional data into fewer dimensions while preserving as much variance (information) as possible. It finds the directions of maximum spread in the data.

  • Use case: Visualise 100-dimensional data in 2D
  • Components: New axes ordered by variance explained
  • Benefit: Removes noise, speeds up training

Python — PCA Dimensionality Reduction 
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import numpy as np

# Customer data: age, income, spending, savings, loans, credit
X = np.array([...])  # shape: (200, 6) — 200 customers, 6 features

# Step 1: Standardise (PCA is scale-sensitive)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Step 2: Reduce from 6D to 2D
pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_scaled)

# How much variance is preserved?
variance = pca.explained_variance_ratio_
print(f"PC1 explains: {variance[0]:.1%}")
print(f"PC2 explains: {variance[1]:.1%}")
print(f"Total: {sum(variance):.1%}")
# Output: PC1 explains: 48.3%
#         PC2 explains: 27.1%
#         Total: 75.4%

# Now visualise the 2D projection
import matplotlib.pyplot as plt
plt.scatter(X_2d[:, 0], X_2d[:, 1], alpha=0.7)
plt.xlabel("Principal Component 1")
plt.ylabel("Principal Component 2")
plt.show()
